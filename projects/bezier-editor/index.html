G-code Bezier Editor

About

G-code is rigid — straight lines and arcs. But sometimes you want a smooth, flowing toolpath. This editor lets you draw bezier curves visually and generates the corresponding G-code in real-time as you drag control points.

Built as a single HTML file (progressive web app), it works offline and on mobile. The 3D version extends the concept to full 3-axis paths with depth control.
