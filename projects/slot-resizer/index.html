Slot Resizer

Resize slots in DXF files while preserving position and orientation. Feed it a DXF, specify the new width, get a resized DXF out.

Python DXF ezdxf NumPy Active

About

When you need to resize slots in a DXF drawing — change the width while keeping the position, orientation, and centerline intact — doing it manually in CAD is tedious and error-prone, especially with angled or 3D slots.

Slot Resizer automates this. Give it a DXF file and a target width, and it fits a plane to the slot geometry, recalculates the arcs and lines at the new width, and outputs a clean resized DXF ready for CAM programming. Supports both 2D and 3D slot geometry with automatic width calculation.

Usage

python slot_resizer.py input.dxf output.dxf --width 2.8
