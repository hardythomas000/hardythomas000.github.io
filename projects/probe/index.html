About

A post-driven probing system for Haas machines with TRT (trunnion/rotary table). The programmer chooses which features to probe before posting in Mastercam — the post processor detects marker operations and outputs safe positioning sequences, Renishaw Inspection Plus macro calls, and correction logic.

Capabilities

  • Bore/boss diameter measurement with automatic tool wear correction
  • Pocket and slot width/length measurement with pass/fail reporting
  • Z datum/face probing with work offset update

How It Works

Marker operations in Mastercam carry parameters in Misc Integers/Reals. The post processor detects these markers and outputs the appropriate Renishaw macro calls (9811, 9812, 9814) with safe positioning, then applies correction logic — either updating tool wear offsets or work coordinate offsets based on the measurement.
