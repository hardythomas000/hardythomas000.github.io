About

A browser-based dashboard for monitoring Haas CNC spindle load in real-time. Built to give machinists and programmers visibility into what's happening inside the machine during cuts — spindle load is one of the best indicators of tool condition, cut quality, and potential problems.

Includes Haas macro integration scripts for data collection and a web frontend for visualization. Designed to run on a shop floor tablet or monitor next to the machine.
