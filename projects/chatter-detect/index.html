About

Chatter (self-excited vibration) is the #1 unsolved practical problem in CNC machining. It damages parts, breaks tools, and wastes time. Experienced machinists can hear it — this project aims to detect it automatically using a cheap microphone or accelerometer.

The concept is an aftermarket, non-invasive system that uses physics-informed deep learning to detect chatter in real-time, before it damages the part or tool. The idea was evaluated by a swarm of 4 AI agents for feasibility.

Currently in research/idea stage — collecting references, evaluating approaches, and building the foundation for a prototype.
