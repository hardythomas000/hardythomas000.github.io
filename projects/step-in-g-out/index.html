About

STEP-in-G-out takes the standard CAD exchange format (STEP) and converts it directly to CNC machining code (G-code). The goal is to eliminate the manual CAM programming step for straightforward parts — load your model, get code you can run on a Haas.

The project includes a Python core engine, a GUI, and reference STEP/G-code files for validation. Research findings from evaluating different approaches to automatic feature recognition and toolpath generation are documented alongside the code.
