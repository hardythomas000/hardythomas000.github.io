Universal Setup Sheet

Browser-first CNC setup sheet system for machine shops. Two versions: programmer's (live editable) and machinist's (read-only export).

Interactive Explainer GitHub

About

Every machine shop needs setup sheets — the document that tells the machinist how to set up and run a job. Most shops use paper, spreadsheets, or clunky legacy software. Universal Setup Sheet is a modern browser-based alternative that works on any device.

Capabilities

  • G-code parsing — supports .nc, .tap, .gcode, .ngc, .eia, and .txt files
  • Tool data import from folder exports and .tooldb files
  • Optional STEP model loading with datum placement
  • Live design controls and print section toggles
  • Export to Excel, printable HTML, and machinist interactive HTML

Two Modes

The Programmer's version is fully editable — the CAM programmer fills in tool data, adds notes, adjusts layout. The Machinist's version is a read-only export the operator uses at the machine. Clean, printable, no accidental edits.
