CNC/CAM

Frame Generator

Automated frame/tab generator for 5-axis CNC workholding. Upload a STEP file, paint constraints, auto-generate support frames with tabs.

Python Three.js FastAPI build123d Early Stage
GitHub

The Problem

Junior engineers manually model support frames in SolidWorks for 5-axis CNC parts. It's rework-prone and blind to CAM programmer constraints — the engineer doesn't know which surfaces need clearance for toolpaths, where tabs should go, or where support is actually needed.

The Solution

Let the CAM programmer define constraints visually on the 3D part. Upload a STEP file, view it in the browser, paint manufacturing annotations (support here, keep clear, no tab), and auto-generate a support frame with tabs. Export as STEP for machining.

The web-based tool uses Three.js with occt-import-js (WASM) on the frontend and Python with build123d/FastAPI on the backend.
