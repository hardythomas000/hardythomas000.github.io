About

I'm a CNC Machinist and CAM Programmer at PWR Performance Products on the Gold Coast, Australia. By day I program Haas and Mazak machines with Mastercam. Recently I've been exploring ideas at work and at home — using Python, HTML, and JavaScript to build tools for CAD/CAM workflows.

That's led to things like browser-based toolpath visualizers, setup sheet generators, interactive G-code editors, and a full SDF-based CAD kernel that AI agents can drive through MCP tools. The thread connecting it all is making CNC machining more accessible and more programmable.
